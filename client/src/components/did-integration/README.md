# D-ID Integration 组件

## 说明

此目录包含与D-ID服务集成相关的组件。这些组件与项目的主要SadTalker功能**完全独立**。

## ⚠️ 重要区别

### SadTalker功能（主要功能）
- **用途**: 基于用户文本输入，AI回复时自动生成说话视频
- **工作流程**: 用户打字 → AI回复 → 自动生成视频（使用管理员设置的默认头像）
- **头像管理**: 只有管理员可以设置默认头像
- **用户权限**: 受试用户只能观看，无法上传头像

### D-ID功能（独立功能）
- **用途**: 实时视频对话功能
- **工作流程**: 用户与AI进行实时视频互动
- **特点**: 独立的第三方服务集成
- **状态**: 当前已隔离，避免与SadTalker功能混淆

## 文件列表

- `DIDAgent.tsx` - D-ID代理组件（已隔离）

## 注意事项

1. **不要将D-ID功能与SadTalker功能混合使用**
2. **受试用户界面应该只使用SadTalker功能**
3. **如需要使用D-ID功能，请确保在独立的上下文中使用**
